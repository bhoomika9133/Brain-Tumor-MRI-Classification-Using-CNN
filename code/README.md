# Code

This folder contains the Jupyter Notebook implementation for the Brain Tumor Detection Framework.

File:

* Brain_Tumor_Project_All_Code_With_Graphs.ipynb

The notebook includes:

* MRI image dataset loading
* Image preprocessing
* CNN model construction
* Model training
* Performance evaluation
* Accuracy and loss graph generation
* Misclassification analysis
* Dropout regularization analysis
* Results visualization
