### Figure Description

The methodology figure illustrates the workflow of the Brain Tumor Detection Framework:

MRI Dataset Collection → Image Preprocessing → Dataset Splitting → CNN Model Construction → Model Training → Performance Evaluation → Tumor Prediction

The accuracy graph illustrates the training and validation performance of the CNN model during training.

### Purpose

These figures are included to visualize the proposed methodology and demonstrate the performance of the Brain Tumor Detection model.
